# 🎬 Top 10 Movies App

A Flask-based web application that allows users to create and manage a list of their top 10 favorite movies using data fetched from an external API.

---

## 🚀 Features

- Search movies using an external API  
- Add movies to a personal list  
- Automatically fetch movie details (poster, description, year)  
- Rate and review movies  
- Dynamic ranking system  

---

## 🛠 Tech Stack

- Python (Flask)  
- SQLAlchemy  
- Flask-WTF  
- Bootstrap  
- Requests  
- TMDB API  

---

## 🧠 What I Learned

- Working with REST APIs and JSON data  
- Backend development with Flask  
- Database design using SQLAlchemy  
- Handling user input with forms  
- Building full-stack web applications
