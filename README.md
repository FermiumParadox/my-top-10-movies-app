# 🎬 Top 10 Movies App

![Screenshot 1](Screenshot_1.png)
![Screenshot 2](Screenshot_2.png)

A Flask-based web application that allows users to create and manage a list of their top 10 favorite movies using data fetched from an external API.

---

## 🚀 Features

* Search movies using an external API
* Add movies to a personal list
* Automatically fetch movie details (poster, description, year)
* Rate and review movies
* Dynamic ranking system

---

## 🛠 Tech Stack

* Python (Flask)
* SQLAlchemy
* Flask-WTF
* Bootstrap
* Requests
* TMDB API

---

## 🧠 What I Learned

* Working with REST APIs and JSON data
* Backend development with Flask
* Database design using SQLAlchemy
* Handling user input with forms
* Building full-stack web applications

## ⚙️ Setup

### 1. Clone the repository

```bash
git clone https://github.com/FermiumParadox/my-top-10-movies-app.git
cd my-top-10-movies-app
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Create a `.env` file

Create a file named `.env` and add:

```env
MOVIE_API_KEY=your_api_key
SECRET_KEY=your_secret_key
```

### 4. Run the app

```bash
python main.py
```
